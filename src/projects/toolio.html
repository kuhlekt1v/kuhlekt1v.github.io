Toolio

Toolio is a mobile-first application designed to streamline the rental, delivery, and return of complete tool rental packages. Tailored for urban DIY enthusiasts who often face space constraints, Toolio offers an affordable, hassle-free way to access high-quality tools without the burden of storage or high costs. Each package includes the essential tools, supplies, and materials necessary for a variety of home improvement or DIY projects.

Key Features

  • Convenient: Toolio offers a modern digital rental experience, allowing users to rent tools with just a few taps on their mobile devices. Tools are delivered directly to the user's home, and once the project is complete, Toolio manages the return process as well.
  • Budget-Friendly: Renting from Toolio is a cost-effective alternative to purchasing expensive tools. Users save money by paying only for what they need when they need itâ€”no need for long-term storage or investment in tools they might only use once or twice.
  • Comprehensive Packages: Toolio provides over 65 unique tool packages that consist of more than 200 tools. These packages are designed to meet the specific needs of different projects, ensuring users have everything required to get the job done efficiently.

Technology Stack

  • Angular Ionic Framework: For building a performant, mobile-friendly, and cross-platform front end interface.
  • Capacitor: For accessing native mobile functionalities like push notifications, camera access, and more.
  • Supabase Cloud Database: A scalable, real-time back end service to manage user data, tool inventories, and orders.
    Note: Supabase utilizes Row Level Security (RLS) to protect user data during database transactions, ensuring that users can only access their own data securely.
  • Supabase Serverless Edge Functions: For secure, server-side operations, such as processing tool rentals and managing deliveries in real time.
  • Stripe API: For secure, efficient payment processing within the app.
